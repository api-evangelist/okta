# OktaThis is a repo for managing machine readable artifacts for the [Okta APIs](https://www.okta.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Okta APIs](https://www.okta.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.okta.com)- [Developer](https://www.okta.com)- [Twitter](https://twitter.com/okta)- [GitHub](https://github.com/okta)The [APIs.json](https://github.com/api-evangelist/okta/blob/master/apis.json) index provides links to all the relevant links for the Okta APIs, including the OpenAPI and Postman if they are present.
